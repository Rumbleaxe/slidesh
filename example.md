# Welcome to slidesh

A dual-target presentation engine.

Write once. Present anywhere.

---

## The Problem

Existing presentation tools lock you into their ecosystem:

- PowerPoint: Proprietary, desktop-only
- Reveal.js: Web-first, CLI is an afterthought
- Marp: Good, but limited customization

What if you could present in the **browser** and **terminal** with the exact same file?

---

## The Solution

slidesh is a compiler, not a converter.

```
Markdown
    ↓
Parser
    ↓
AST (Abstract Syntax Tree)
    ↓
Renderer (Web or CLI)
```

Same source. Two targets. Zero compromise.

---

## Features

- **Dual Runtime**: Browser + Terminal
- **Arrow Keys**: Navigate in both environments
- **Markdown Native**: Standard Markdown syntax
- **Minimal**: No plugins. No themes. No magic.

Start presenting with slidesh today!
